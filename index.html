3 Lagenmodel.

Het 3 lagen model bestaat uit de fysieke laag, logische laag en toepassingen laag. Door middel van deze verschillende lagen kunnen de hardware en software van een computer samenwerken en de computer laten werken. Specifieke uitleg over wat de lagen doen en waar ze uit bestaan kan je ook op de site vinden.

Fysieke laag


De fysieke laag bestaat veel verschillende dingen, deze dingen noem je de hardware. Hardware zijn alle onderdelen die je met je eigen hand kan aanraken, vandaar ook de naam fysieke laag. Hieronder staan een aantal hardware met uitleg van hun functie.



Logische laag

De logische laag van een apparaat bestaat uit software die ervoor zorgt dat toepassingen op het apparaat uitgevoerd kunnen worden. Deze software maakt het mogelijk om apps en andere software programma’s uit te voeren. De grootste voorbeelden zijn IOS en Android.

Toepassingen laag

Bij een smartphone kun je apps installeren zoals bijvoorbeeld: TikTok, Instagram, Snapchat. Applicaties, ook wel apps, zijn de software waaruit de toepassingen laag bestaat. Denk hierbij aan alle apps die je kunt downloaden in de app- of play store.


Moederbord.

Bij dit kopje geven we informatie over het moederbord.

Het moederbord vormt het kloppend hart van je computer. Het is een printplaat die alle componenten van stroom voorziet en met elkaar laat communiceren. Je processor, werkgeheugen, geluidskaart, videokaart en andere onderdelen zijn dus allemaal aangesloten op het moederbord.


Processor en Geheugen.

Kernapperatuur van een computer

Het ram-geheugen staat voor random access memory. De processor kan niet zonder ram geheugen en andersom kan het het ram geheugen niet zonder de processor, ook wel CPU genoemd. Het RAM geheugen heeft als functie het tijdelijk bewaren van je besturingssysteem en programma’s die je open hebt staan. Het CPU zorgt ervoor dat alles wordt uitgevoerd. Als je een CPU hebt maar geen RAM geheugen heb je niks aan je CPU, deze hebben elkaar namelijk nodig om een taak of instructie uit te voeren.


Besturingsysteem.

Het besturingssysteem zorgt ervoor dat er duidelijke communicatie is tussen de applicatiesoftware en de hardware. Bij alle taken die worden uitgevoerd maken de applicaties gebruik van besturingssystemen, bijvoorbeeld als je een bestand verwijderd. Voorbeelden van besturingssystemen zijn:
1. Apple
2. Chrome
3. Windows
4. Android
Het besturingssysteem kent drie functies:
1. Beheer van randapparatuur
2. Bestandsbeheer
3. Taakbeheer
Het besturingssysteem zorgt voor aansturing van de hardware, het enige probleem is dat het besturingssysteem zelf geen kennis heeft dus wordt er gebruik gemaakt van drivers. Ook zorgt het besturingssysteem ervoor dat alle taken eerlijk verdeeld worden over de CPU.
